how to run the backend with Docker?

for windows/MacOs make sure you have Docker Desktop installed and signed in
In Terminal:
docker login
(give your credentials username and password to login)
docker-compose build
(this builds containers)
docker-compose up
(this starts the server)

Now you can make requests for testing


how to run the backend with Node?

In Terminal:
npm start

If you make any requests to CodeEditor , then either make it after using Docker or if you are using npm then make sure that your local has the language installed
Pyhton-pip
c++-MingiW64
Java-JDK
